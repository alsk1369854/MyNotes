#### 快捷鍵

- 上方工具列 Xcode => Preferences => Key Binding

| 類型               | 符號    | 意思            |
| ---------------- | ----- | ------------- |
| Range            | a..<b | a 到小於 b 的所有數字 |
| ClosedRange      | a...b | a 到 b 的所有數字   |
| PartialRangeFrom | a...  | 所有等於大於 a 的數字  |
| PartialRangeFrom | ...a  | 所有小於或等於 a 的數字 |
| PartialRangeFrom | ..<a  | 所有小於 a 的數字    |
